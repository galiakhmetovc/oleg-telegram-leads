# KEYWORDS.md — Словари ключевых слов

Ключевые слова используются в `keyword_scanner.py` для первого этапа фильтрации.
Поиск по **отдельным словам** (word boundaries) — никаких частичных совпадений внутри слов.
Многословные (2+ слова) проверяются через fuzz.WRatio с порогом 84.

## Принципы составления

1. **Каждое слово — в нижнем регистре**, без падежей (scanner сам нормализует)
2. **Только однословные термины** —scanner ищет их как отдельные слова в тексте
3. **Многословные** через `_` — scanner использует fuzz для вариаций (ед/мн число)
4. **Бренды** — только те, что реально используются в РФ (рынок Олега)
5. **Короткие слова** (< 4 символов) — с осторожностью, чтобы не было false positives
6. **ОПАСНО**: Не добавляйте слова, которые встречаются в бытовом контексте:
   - ❌ "насос" — есть в фразах "насос для накачки шин"
   - ❌ "сеть" — есть в "сеть магазинов", "социальная сеть"
   - ❌ "замок" — есть в "висячий замок", "замок на двери шкафа"
   - ❌ "датчик" — слишком широкое слово, встречается часто вне контекста

## Как пополнять словари

1. Добавляйте слова по мере обнаружения новых брендов/терминов в чатах
2. Не добавляйте слишком короткие слова (< 4 символов) без контекста
3. Бренды добавляйте с основными опечатками (2-3 варианта)
4. Периодически просматривайте false positives и корректируйте threshold или убирайте шумные слова
